# \[JavaScript] Operators

## `typeof`
```javascript
>   str = '안녕하세요'
>   if (typeof str == 'string')
    console.log('문자열');
<   '문자열'
```
변수, Literal 상수의 타입을 문자열로 반환한다.

## `!`, 부정연산자
```javascript
>   !true
<   false

>   !false
<   true

>   !undefined
<   true

>   !null
<   true
```
undefined 값을 boolean 값으로 출력한다
